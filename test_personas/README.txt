TEST PERSONAS FOR CAREER COUNSELOR CHATBOT
==========================================

These 5 personas represent diverse student profiles to test the chatbot UX.
Each file contains a simulated response to the initial prompt.

PERSONA SUMMARY
---------------

1. Marcus (01_marcus_stem_texas.txt)
   - STEM-focused mechanical engineering senior
   - Strengths: Problem-solving, CAD, hands-on work
   - Weakness: Technical writing
   - Location: Texas only (Houston/Austin/Dallas)
   - TEST FOCUS: Engineering careers, state-specific data for Texas

2. Sophia (02_sophia_creative_flexible.txt)
   - Creative communications/design graduate
   - Strengths: Visual design, software, creative writing
   - Weakness: Numbers/business side
   - Location: Flexible, wants creative cities
   - TEST FOCUS: Arts/design careers, comparing multiple cities

3. James (03_james_healthcare_midwest.txt)
   - Pre-med student reconsidering path
   - Strengths: Patient interaction, hands-on labs
   - Weakness: Memorization, competitive pressure
   - Location: Midwest (MI, OH, IL, MN)
   - TEST FOCUS: Healthcare careers without MD, alternative paths

4. Aisha (04_aisha_business_data_northeast.txt)
   - Finance/stats major, data-focused
   - Strengths: Quantitative analysis, Python/SQL
   - Weakness: Big-picture thinking, networking
   - Location: NYC, Boston, DC
   - TEST FOCUS: Finance + data analytics, high-growth careers

5. Tyler (05_tyler_trades_rural.txt)
   - Non-traditional, construction background
   - Strengths: Hands-on learning, mechanical aptitude
   - Weakness: Classroom learning
   - Location: Rural Idaho
   - TEST FOCUS: Skilled trades, no-degree paths, rural job markets

TESTING CHECKLIST
-----------------
For each persona, verify the chatbot:
[ ] Uses appropriate search/filter tools
[ ] Pulls state-specific data when location mentioned
[ ] Suggests careers matching their strengths
[ ] Acknowledges their constraints (education, location)
[ ] Provides specific salary and growth data
[ ] Offers to explore alternatives or dig deeper
